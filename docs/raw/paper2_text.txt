Conclusion In this paper, we  have  presented  the  ECRVR-MVEL   approach  to design an interpretable   methodology  for code readability analysis that  improves   the  quality of  code.  Primarily , the proposed model preprocesses the input code that  contains   four   stages   like  tokenization, removal  of  comments or normalization of white space and detecting languages.  Furthermore , the CodeBERT is  employed for  transform ing  raw code into vector representations,  allowing   effective  text embeddings for  a  classification model. For   classification, a weighted majority voting ensemble classification  incorporates   GCN ,  DBN  and  BiTCN   for classifying  code readability into high, medium and low levels.  Additionall y , the Nadam is  implemented  to  boost  the  execution  of  the  code readability classification model.  Ultimately , the LIME is  applied  to visualize the interpretability and transparency of  the  model.  A n  extensive experimental outcome  is  carried out  to assure the significance of the ECRVR-MVEL  approach . A  brief comparison o f outcomes  reported the promising  solutions  of the ECRVR-MVEL model  through  other existing models.
