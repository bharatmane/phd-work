The EESQA-DELMOA model aims to assess software quality by analy z ing the developer ' experience levels on code reliability and maintainability. Initially, the data pre-processing stage applies min-max normalization method to clean, transform, and organize raw data into a structured format. In addition, the proposed EESQA-DELMOA model implements BAHB technique for the selection of feature process to select the most relevant features for enhancing model performance. Followed by, the SSNN method has been deployed for the classification method. Eventually , the parameter tuning process is implemented through AMBOA to enhance the classification performance of the SSNN classifier. The experimental evaluation of the EESQA-DELMOA model occurs using benchmark dataset.
